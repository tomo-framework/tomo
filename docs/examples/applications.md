## Next.js Chat App
A full chat application with Tomo tool integration.

### Key Files
- `app/page.tsx`: Chat interface
- `app/api/chat/route.ts`: AI handler
- `lib/tomo-mcp-client.ts`: MCP client

### Features
- Real-time chat
- Tool calling
- Streaming responses
